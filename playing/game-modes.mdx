XMage ships with one of the broadest format catalogs of any Magic: The Gathering client, covering everything from kitchen-table Freeform to tournament-legal Standard and the full Commander family. The server registers 17 distinct game types and 52 deck validators, all with full rules enforcement — deck legality is checked before a game starts and game rules are enforced throughout every match. Whether you want a two-player competitive duel, a 10-player Commander pod, or a nostalgic Old School 93/94 game, there is a game type and a matching deck validator waiting for you.

## Game Types

Game types define the *structure* of a match — how many players, which zone rules apply (commander tax, etc.), and which special rules are active. Every game type listed below is registered in the server's `config.xml` and is selectable when creating a new table.

### Duel (Two-Player) Formats

| Game Type | Description |
|---|---|
| **Two Player Duel** | Classic 1v1 match. The baseline format for most constructed play. |
| **Commander Two Player Duel** | Commander rules in a 1v1 setting; uses standard Commander ban list. |
| **Tiny Leaders Two Player Duel** | Commander-style 1v1 with a legendary creature of CMC ≤ 3; deck max CMC ≤ 3. |
| **Canadian Highlander Two Player Duel** | 100-card singleton 1v1 using the Canadian Highlander points system. |
| **Freeform Commander Two Player Duel** | Commander rules with no official ban list — bring anything. |
| **Custom Pillar of the Paruns Two Player Duel** | Variant duel using the Pillar of the Paruns custom rule set. |
| **Oathbreaker Two Player Duel** | Oathbreaker rules in a 1v1 setting. |
| **Brawl Two Player Duel** | Brawl (Standard-legal commander variant) in a 1v1 setting. |
| **Momir Basic Two Player Duel** | Each turn you may pay X generic mana and discard a card to put a random creature of CMC X into play, 1v1. |

### Multiplayer (Free For All) Formats

| Game Type | Description |
|---|---|
| **Free For All** | Multiplayer game with no special zone rules; each player starts at 20 life. |
| **Commander Free For All** | Commander rules for 3–10 players; 40 starting life. The most popular mode. |
| **Penny Dreadful Commander Free For All** | Commander with Penny Dreadful pricing restrictions on the card pool. |
| **Freeform Commander Free For All** | Commander rules, no ban list, multiplayer. |
| **Freeform Unlimited Commander** | Commander rules with no singleton restriction and no ban list. |
| **Oathbreaker Free For All** | Oathbreaker rules for multiplayer pods. |
| **Brawl Free For All** | Brawl rules in a multiplayer pod. |
| **Momir Basic Free For All** | Momir Basic variant for multiple players. |

## Choosing the Right Game Type and Validator

When you create a new table you choose *both* a **game type** (defines the match structure) and a **deck type** (validates the deck you bring). They must be compatible — for example, use **Commander Two Player Duel** as the game type and **Variant Magic - Commander** as the deck type for a standard 1v1 Commander game.

<Note>
  Custom sets like the Star Wars fan set are fully implemented in XMage. Select **Block Constructed Custom - Star Wars** as the deck type to play with those cards in a dedicated constructed environment.
</Note>
