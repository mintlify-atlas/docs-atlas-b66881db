XMage lets you play Magic: The Gathering against human opponents over a network or against computer AI opponents — all with complete, automatic rules enforcement. It supports over **31,000 unique cards** and more than **91,000 reprints** across hundreds of sets, plus custom sets like Star Wars. Every regular set has nearly all cards implemented.

## What is XMage?

XMage is a client-server Magic: The Gathering simulator written in Java. The server enforces the complete Magic ruleset automatically — you never have to manually resolve triggers, track priority, or look up rules interactions. Both the client and server are free and open source under the MIT license.

## Get Started in 3 Steps

<Steps>
  <Step title="Download the Launcher">
    Get the latest XMage launcher and app files from [xmage.today](http://xmage.today/). Unpack the archive to any folder (e.g. `D:\games\xmage`).
  </Step>
  <Step title="Install Java">
    XMage requires Java 8 or later. Download it from [java.com](https://java.com/) if you don't have it installed.
  </Step>
  <Step title="Connect and Play">
    Launch XMage, connect to a public server or start a local server, build or import a deck, and start a game.
  </Step>
</Steps>
